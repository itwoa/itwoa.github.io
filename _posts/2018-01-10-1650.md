## Mysql 常用命令汇总

#### ● Mysql 重置密码

`mysql
    mysqladmin -uroot -p'root' password "123456"
`

#### ● 添加索引

`mysql
    ALTER TABLE  `cool_personinfo`  ADD INDEX ( `currentt` );
`

#### ● 删除索引

`mysql
    ALTER TABLE `cool_personinfo` DROP INDEX `shopp2_2`;
`

#### ● 修改表的存储类型

`
    ALTER TABLE `cool_account` ADD FULLTEXT (`namee`)
    //存储类型修改为MyISAM
`
